## Data Type

### 📌 String
```js
const name = "Jiyu";
alert(`Hello, ${name}!`); // Hello, Jiyu!
```

### 📌 Boolean
```js
const what = ture;

let isGreater = 4 > 1;
alert(isGreater); // true
```

### 📌 Number
정수만을 표기하기 위한 자료형은 없음
```js
const num = 666;
const float = 51.5;
```

### 📌 Null
알 수 없는 값이거나 비어있는 값
```js
const age = null;
```

### 📌 Undefined
값이 할당되지 않은 상태
```js
const age;
alert(age); // undefined
```
